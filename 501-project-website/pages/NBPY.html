Naive Bayes in python

In Python, the naive bayes is computed using a text dataset. The tweets are based on the hashtags "climatechange" and "savetheplanet." The goal of collecting this text data was to learn about people's perspectives on how important climate change is in today's world. To get a quick overview of the data, a wordcloud of both the hashtags has been made.

Model builiding

Code for the model building

The dataset is divided into training and testing sets before the model is built. The split ratio in the training set is 0.75 and in the testing set it is 0.25. Three distinct naive bayes models are developed. The hypertuning of various parameters distinguishes the Naive Bayes models. Specifically, the alpha values.

Model 1

The first naive bayes model was developed. The hyperparameter in this model is alpha = 1. The accuracy of this model is 89%.

The snapshot of model 1

Model 2

The first naive bayes model was developed. The hyperparameter in this model is alpha = 0. The accuracy of this model is same as model 1 in this case, that is 89%.

The snapshot of model 2

Model 3

The first naive bayes model was developed. The hyperparameter in this model is alpha = 5. The accuracy of this model 92%.

The snapshot of model 3

Conclusion

The goal was to find people's opinion on today's climate change using recent twitter data. The Naive Bayes model classified tweets generated by Twitter into different hashtag classes. The models' accuracy is 89% and 92%, respectively. The precision is quite high.
